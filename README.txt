# SIH Electric Fence Monitoring — SPA Dashboard

A **Smart India Hackathon project** for monitoring electric fences using **Drone + IoT + RFID** data.  
This dashboard provides **real-time visualization**, **alerts**, and **analytics** to enhance security.

---

## 🚀 Features
- 📍 **Interactive Map** (Leaflet.js) with fence markers (Authorized ✅ / Unauthorized ❌)
- 📊 **Charts & Analytics** (Chart.js: Pie, Bar, Line, Timeline)
- ⚡ **SPA (Single Page App)** navigation with sidebar
- 🔔 **Alerts Log** with export options (CSV, PDF — demo)
- 🎛 **Settings** to add RFID whitelist
- 🌗 **Light / Dark Mode Toggle** for presentation
- 🔄 **Live Refresh Button** (mock data demo)

---

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Charts**: [Chart.js](https://www.chartjs.org/)
- **Maps**: [Leaflet.js](https://leafletjs.com/)
- **SPA Routing**: Custom JS (hash-based)
