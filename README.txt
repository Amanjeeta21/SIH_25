🚀 SIH 2025 Project

📌 Overview
This project was developed as part of **Smart India Hackathon (SIH) 2025**, a nationwide innovation challenge focused on solving real-world problems. The goal of our solution is to build a scalable and user-centric system that addresses practical challenges using modern technologies.

The project is based on solving a real-world problem provided under SIH, focusing on delivering an efficient, scalable, and impactful solution with a strong emphasis on usability and performance.
